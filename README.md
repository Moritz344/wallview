# Features 
- [x] view your local wallpapers
- [x] browse wallhaven wallpapers
- [x] download wallhaven wallpapers
- [x] change local wallpaper path
- [x] show wallpaper in folder


# Running Locally
```bash
git clone https://github.com/Moritz344/wallview.git
cd wallview/
npm install
npm start
```
